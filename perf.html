How I Styled My Pages

I kept my CSS simple because I wanted the layout to stay clean and easy to follow. I used soft colors, rounded edges, and a basic hover effect for the navigation menu. The design isn’t overdone, but it feels balanced and readable.

Instead of fancy transitions, I focused on making sure each element had space to breathe. I added box shadows to make the content stand out and used consistent headings and colors across all pages. The design may be basic, but it represents where I’m at in my learning.

This site is a class project for IT 3203.
