Core Ideas I Focused On

  • HTML: The structure and foundation of any website.
  • CSS: The styling layer that controls colors, layout, and spacing.
  • Semantics: Using meaningful tags to improve readability and accessibility.
  • Responsiveness: Making sure a website adjusts to any screen size.
  • Design Simplicity: Keeping things minimal but effective.
  • Accessibility: Ensuring all users can easily navigate and read the content.
